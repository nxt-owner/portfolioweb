Singlish to Sinhala Converter Chrome Extension

Convert English phonetic typing (Singlish) to Sinhala script instantly with our easy-to-use Chrome extension. Perfect for anyone who wants to type in Sinhala without learning a complex keyboard layout.

Download Extension

Key Features

Easy Phonetic Typing

Type Sinhala words using English letters that sound similar (e.g., "ayubowan" becomes "ආයුබෝවන්").

Works Anywhere

Use it on any website - Facebook, WhatsApp Web, Gmail, and more.

Instant Conversion

See your Singlish text convert to Sinhala as you type with no delay.

Toggle On/Off

Easily enable or disable the conversion with a simple click.

Installation Guide

Download the Extension

Click the download button above to get the .crx file or .zip file of the extension.

Unzip The Downloaded File

Unzip your downloaded zip and save it in a location you want

Open Chrome Extensions

In your Chrome browser, go to chrome://extensions in the address bar.

Enable Developer Mode

Toggle the "Developer mode" switch in the top right corner to ON.

Load the Extension

Click "Load unpacked" and select the extension folder you downloaded, or drag and drop the .crx file into the extensions page.

Pin the Extension

Click the puzzle icon in Chrome's toolbar and pin the "Singlish to Sinhala" extension for easy access.

How to Use

Activate the Extension

Click the extension icon in your toolbar to activate it (the icon will turn blue when active).

Start Typing

Go to any text field on any website and start typing in Singlish (English letters that sound like Sinhala words).

See Instant Conversion

Your text will automatically convert to Sinhala script as you type.

Toggle as Needed

Click the extension icon again to toggle the conversion off when you need to type in English.

Troubleshooting

  • Extension not working?

    Try refreshing the page or restarting Chrome. Make sure the extension is enabled in your extensions list.

  • Conversion not accurate?

    Some words may have multiple Sinhala equivalents. Try alternative phonetic spellings.

  • Not working on a specific site?

    Some websites have complex text fields that may not work with the extension. Try a different browser or site.
